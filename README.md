# 🌤️ Weather Widget App

A responsive, real-time weather application built with **React** and **Material-UI**, powered by the **OpenWeather API**. Users can search for any city and instantly view current weather conditions including temperature, humidity, and weather description.

🔗 **Live Demo:** [https://nancydhattarwal.github.io/Weather-widget-app/](https://nancydhattarwal.github.io/Weather-widget-app/)

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Displays real-time temperature, humidity, and weather condition
- 🎨 Clean, modern UI built with Material-UI components
- ⚡ Fast build and development with Vite
- 📱 Responsive design — works on all screen sizes
- 🌐 Deployed on GitHub Pages

---

## ⚙️ Getting Started

### Prerequisites
- Node.js (v16 or above)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Nancydhattarwal/Weather-widget-app.git

# Navigate to project directory
cd Weather-widget-app

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

### Deploy to GitHub Pages

```bash
npm run deploy
```

---

## 🔑 Environment Setup

This app uses the **OpenWeather API**. To run it locally:

1. Get a free API key from [https://openweathermap.org/api](https://openweathermap.org/api)
2. Create a `.env` file in the root directory:

```env
VITE_WEATHER_API_KEY=your_api_key_here
```
